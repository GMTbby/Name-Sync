/b/ Name Sync

Shares your name with other posters on 4chan's forced anon boards (/b/, /q/ and /soc/)

Install 4chan X (required)

Install /b/ Name Sync

Features

  • Share names online
  • Assign names to Anonymous posters
  • Hide ID's next to poster names
  • Support for non-secure tripcodes

Before Installing

By installing this script you understand that the following information is temporarily stored on a remote server:

  • Your name field (tripcode passwords are not stored)
  • Your email field
  • Your subject field
  • A unique identifier based on your IP address
  • Thread number you are replying to
  • Post number of the post you have submitted
  • A timestamp of when the post was made

There is a button to remove this stored information in settings.

Supported Browsers

The latest stable releases of these browsers are supported.

  • Chrome
  • Firefox with Greasemonkey
  • Opera Mobile or Opera

Troubleshooting

  • Make sure cookies are enabled
  • Update Name Sync
  • Update 4chan X
  • Update your browser
  • Allow the 'milkyis.me' domain in your privacy and proxy extension settings

If these do not work, make an issue here
