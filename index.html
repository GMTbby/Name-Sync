/b/ Name Sync

Shares your name with other posters on /b/. Also allows you to assign names to Anonymous posters.

Please read the information on this page before installing.

Install Now

Features

  • Share names online
  • Assign names to Anonymous posters
  • Hide ID's next to poster names
  • Support for non-secure tripcodes

Before Installing

Install 4chan X.

The latest version requires Mayhem 4chan X 2.33.1 or later.

Posting

  • Your name, subject and email fields in the QR box are shared (or as specified in options)
  • Only posts with images are shared

Supported Browsers

  • Chrome/Chromium
  • Firefox/Nightly with Scriptish
  • Opera Mobile/Opera 12
Greasemonkey, Scriptify and Safari are unsupported
