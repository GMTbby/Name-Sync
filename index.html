/b/ Name Sync

Shares your name with other posters on /b/. Also allows you to assign names to Anonymous posters.

Install now

Features

  • Share names online
  • Assign names to Anonymous posters
  • Hide ID's next to poster names
  • Support for non-secure tripcodes

Before Installing

Install 4chan x

Posting

  • Your name, subject and email fields in the quick reply box are shared (or as specified in options)

Supported Browsers

The latest stable releases of these browsers are supported.

  • Chrome
  • Firefox with Scriptish
  • Opera Mobile/Opera
