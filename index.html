/b/ Name Sync

Shares your name with other posters on /b/. Also allows you to assign names to Anonymous posters.

Install now

Features

  • Share names online
  • Assign names to Anonymous posters
  • Hide ID's next to poster names
  • Support for non-secure tripcodes

Before Installing

By installing this script you understand that the following information is stored on a remote server for three hours before being cleared:

  • Your name field (If your name field contains a tripcode, the result is stored, but not the password)
  • Your email field
  • Your subject field
  • A unique identifier based on your IP address
  • Thread number you are replying to
  • Post number of the post you have submitted
  • A timestamp of when the post was made

Do not install this script if you do not want this information temporarily stored.

Install 4chan x
