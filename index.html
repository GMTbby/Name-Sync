4chan X Name Sync

Enables names on 4chan's forced anon boards /b/, /q/ and /soc/

  • Share names online bypassing forced anonymity
  • Assign names to Anonymous posters
  • Hide ID's next to poster names
  • Support for tripcodes

4chan X is required.

Install 4chan X v3 here (recommended) or v2 here.

  • Try the following if you are having issues:

    • Verify you are running 4chan X v3 and 4chan X Name Sync v4
    • Update 4chan X
    • Update 4chan X Name Sync
    • Update your browser manually
    • Allow the 'www.milkyis.me' domain in any privacy and proxy add-on settings
  • Read this before installing:

    • Your name, email, subject fields and a unique ID based on your IP address are stored online when you submit a post
    • Tripcode results are stored online, tripcode passwords are not
